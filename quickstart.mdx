This guide walks you through installing the SDK, configuring credentials, and making real API calls against Amazon S3.

<Note>
  Run integration tests only against non-production AWS accounts. The SDK creates and deletes real AWS resources during testing, and charges may apply.
</Note>

<Steps>
  <Step title="Install the S3 package">
    The SDK is distributed as modular NuGet packages — one per service. Add the `AWSSDK.S3` package to your project:

    <CodeGroup>

    ```bash .NET CLI
    dotnet add package AWSSDK.S3
    ```

    ```powershell NuGet Package Manager
    Install-Package AWSSDK.S3
    ```

    </CodeGroup>

    `AWSSDK.Core` is pulled in automatically as a transitive dependency.
  </Step>

  <Step title="Configure credentials">
    The SDK uses a credential provider chain that automatically checks several sources in order. For local development, the simplest approach is to configure the AWS CLI:

    ```bash
    aws configure
    ```

    The CLI prompts you for your AWS Access Key ID, Secret Access Key, default region, and output format. Credentials are stored in `~/.aws/credentials`.

    Alternatively, you can supply credentials through environment variables:

    ```bash
    export AWS_ACCESS_KEY_ID=your-access-key-id
    export AWS_SECRET_ACCESS_KEY=your-secret-access-key
    export AWS_REGION=us-east-1
    ```

    <Tip>
      When running on AWS infrastructure (EC2, ECS, Lambda), you do not need to configure credentials manually. The SDK automatically retrieves temporary credentials from the instance or task IAM role.
    </Tip>
  </Step>

  <Step title="Create an S3 client and list buckets">
    Create an `AmazonS3Client` and call `ListBucketsAsync` to verify your credentials and connection:

    ```csharp
    using Amazon.S3;
    using Amazon.S3.Model;

    var s3Client = new AmazonS3Client(Amazon.RegionEndpoint.USEast1);

    ListBucketsResponse response = await s3Client.ListBucketsAsync();

    foreach (var bucket in response.Buckets ?? [])
    {
        Console.WriteLine($"{bucket.BucketName}  (created {bucket.CreationDate:yyyy-MM-dd})");
    }
    ```

    `AmazonS3Client` picks up credentials from the provider chain automatically. To target a different region, replace `USEast1` with the appropriate `RegionEndpoint` value, or set the `AWS_REGION` environment variable.
  </Step>

  <Step title="Upload a file to S3">
    Use `PutObjectAsync` to upload a file to an existing bucket:

    ```csharp
    using Amazon.S3;
    using Amazon.S3.Model;

    var s3Client = new AmazonS3Client(Amazon.RegionEndpoint.USEast1);

    const string bucketName = "my-example-bucket";
    const string keyName    = "uploads/hello.txt";
    const string filePath   = "/tmp/hello.txt";

    await File.WriteAllTextAsync(filePath, "Hello from the AWS SDK for .NET!");

    var putRequest = new PutObjectRequest
    {
        BucketName  = bucketName,
        Key         = keyName,
        FilePath    = filePath,
        ContentType = "text/plain"
    };

    PutObjectResponse putResponse = await s3Client.PutObjectAsync(putRequest);
    Console.WriteLine($"Upload HTTP status: {(int)putResponse.HttpStatusCode}");
    ```
  </Step>

  <Step title="Download the file from S3">
    Use `GetObjectAsync` to retrieve the object you just uploaded:

    ```csharp
    using Amazon.S3;
    using Amazon.S3.Model;

    var s3Client = new AmazonS3Client(Amazon.RegionEndpoint.USEast1);

    const string bucketName    = "my-example-bucket";
    const string keyName       = "uploads/hello.txt";
    const string downloadPath  = "/tmp/hello-downloaded.txt";

    var getRequest = new GetObjectRequest
    {
        BucketName = bucketName,
        Key        = keyName
    };

    using GetObjectResponse getResponse = await s3Client.GetObjectAsync(getRequest);
    await getResponse.WriteResponseStreamToFileAsync(downloadPath, append: false, CancellationToken.None);

    Console.WriteLine(await File.ReadAllTextAsync(downloadPath));
    // Output: Hello from the AWS SDK for .NET!
    ```
  </Step>
</Steps>

## Complete example

The following self-contained program combines all four operations — list, upload, download, and verify:

```csharp
using Amazon.S3;
using Amazon.S3.Model;

const string bucketName   = "my-example-bucket";   // must already exist
const string keyName      = "quickstart/hello.txt";
const string region       = "us-east-1";

using var s3Client = new AmazonS3Client(Amazon.RegionEndpoint.GetBySystemName(region));

// 1. List buckets
Console.WriteLine("Buckets:");
var listResponse = await s3Client.ListBucketsAsync();
foreach (var bucket in listResponse.Buckets ?? [])
    Console.WriteLine($"  {bucket.BucketName}");

// 2. Upload
var putRequest = new PutObjectRequest
{
    BucketName  = bucketName,
    Key         = keyName,
    ContentBody = "Hello from the AWS SDK for .NET!",
    ContentType = "text/plain"
};
await s3Client.PutObjectAsync(putRequest);
Console.WriteLine($"\nUploaded s3://{bucketName}/{keyName}");

// 3. Download
var getRequest = new GetObjectRequest { BucketName = bucketName, Key = keyName };
using var getResponse = await s3Client.GetObjectAsync(getRequest);
using var reader = new StreamReader(getResponse.ResponseStream);
string content = await reader.ReadToEndAsync();
Console.WriteLine($"Downloaded content: {content}");
```
