The AWS SDK for .NET is distributed as a set of modular NuGet packages — one per AWS service, plus a shared core package. You only install the packages for the services your application uses.

All official packages are published under the [awsdotnet](https://www.nuget.org/profiles/awsdotnet) NuGet profile. There are **400+ service packages** available, covering the full breadth of AWS services.

## Installing packages

<Tabs>
  <Tab title=".NET CLI">
    ```bash
    dotnet add package AWSSDK.S3
    dotnet add package AWSSDK.DynamoDBv2
    ```
  </Tab>
  <Tab title="Package Manager Console">
    ```bash
    Install-Package AWSSDK.S3
    Install-Package AWSSDK.DynamoDBv2
    ```
  </Tab>
  <Tab title="PackageReference">
    ```xml
    <ItemGroup>
      <PackageReference Include="AWSSDK.S3" Version="4.*" />
      <PackageReference Include="AWSSDK.DynamoDBv2" Version="4.*" />
    </ItemGroup>
    ```
  </Tab>
</Tabs>

You do not need to add `AWSSDK.Core` explicitly — NuGet resolves it as a transitive dependency of every service package.

## Extension packages

These packages add integration capabilities beyond the base service clients.

| Package | Description |
|---------|-------------|
| `AWSSDK.Extensions.NETCore.Setup` | ASP.NET Core dependency injection integration. Adds `AddAWSService<T>()` to `IServiceCollection`, enabling you to configure SDK clients via `appsettings.json` and inject them into your services. |
| `AWSSDK.Extensions.CrtIntegration` | AWS Common Runtime (CRT) integration. Required for S3 Multi-Region Access Points, S3 Object Integrity checksums, and EventBridge Global Endpoints (which require AWS Signature V4a). |
| `AWSSDK.Extensions.CognitoAuthentication` | Cognito SRP (Secure Remote Password) authentication. Simplifies the challenge-response authentication flow for Cognito User Pools. |

### ASP.NET Core integration example

```csharp
// Program.cs
builder.Services.AddDefaultAWSOptions(builder.Configuration.GetAWSOptions());
builder.Services.AddAWSService<IAmazonS3>();
builder.Services.AddAWSService<IAmazonDynamoDB>();
```

```json
// appsettings.json
{
  "AWS": {
    "Region": "us-east-1",
    "Profile": "my-profile"
  }
}
```
